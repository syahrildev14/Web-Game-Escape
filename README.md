# 🚀 Fullstack Web App (React + Express + MongoDB)

## 🛠 Tech Stack

**Frontend**
- React.js Vite
- Axios
- Tailwind CSS
- Sweetalert2
- React Icon

**Backend**
- Node.js
- Express.js
- JWT Authentication

**Database**
- MySql

---

## ⚙️ Installasi
Frontend
```bash
cd frontend
npm install
npm run dev

```

Backend
```bash
cd backend
npm install
npm run dev
